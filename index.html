Blockchain

The bitcoin blockchain is a public ledger that records bitcoin transactions.[41] It is implemented as a chain of blocks, each block containing a cryptographic hash of the previous block up to the genesis block[c] in the chain.

A network of communicating nodes running bitcoin software maintains the blockchain.[42]: 215–219  Transactions of the form payer X sends Y bitcoins to payee Z are broadcast to this network using readily available software applications.
